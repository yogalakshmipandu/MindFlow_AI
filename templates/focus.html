👁️ Drowsy Alerts

Stay awake while studying - AI powered detection

How to use Drowsy Detection:

  1. Open a new terminal/command prompt
  2. Navigate to the Backend folder: cd c:\Users\nagas\Desktop\Backend
  3. Run the drowsy detection script: python drowsy_detection.py
  4. A camera window will open detecting your eyes
  5. If you look drowsy, an alarm will sound!
  6. Press q in the camera window to quit

Note: Keep the terminal window open while using drowsy detection.
